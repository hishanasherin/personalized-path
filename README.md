## PERSONALIZED PATH

 > Personalized Path is an AI-powered learning roadmap assistant built using IBM watsonx.ai. It interacts with users to understand their interests, skill level, and career goals, then generates a tailored step-by-step learning path with recommended tools, resources, and courses. The roadmap adapts dynamically based on the user’s progress, ensuring focused and goal-oriented learning.

## PROBLEM STATEMENT 

> Many learners struggle to choose the right learning path due to the overwhelming number of online resources and lack of personalized guidance. Existing platforms often provide generic recommendations that fail to align with individual goals, interests, and skill levels. There is a need for an intelligent system that can assess user preferences and progress, then generate a customized, adaptive learning roadmap.


## PR0POSED SOLUTION

> An AI agent built using IBM watsonx.ai that provides reliable, personalized learning roadmaps based on user interests, skill levels, and career goals. With adaptive recommendations and curated resources, it ensures learners receive grounded, goal-oriented guidance for skill development.


## TECHNOLOGIES USED

> IBM watsonx.ai Studio 
IBM Granite Foundation Model (LLM)
Vector Index for Retrieval-Augmented Generation (RAG) 
PDF Documents from Curated Learning Sources 
NLP (Natural Language Processing)
IBM Cloud Object Storage

## WOW Factors
  > Uses RAG to fetch learning materials from trusted educational PDFs and online sources.
    Built entirely on IBM Cloud using watsonx.ai tools.

    Future-ready with multilingual support for global learners.
    Bridges the gap between generic online courses and personalized mentorship.
    Provides safe, reliable, and goal-oriented learning guidance.

## Key Features
  > Document-based roadmap creation via Vector Index.
   Powered by IBM Granite LLM for natural language understanding.
   Built-in NLP to understand and refine user goals.
   Responds politely to off-topic questions with redirection.
   Covers multiple learning domains (Web Dev, AI/ML, Cybersecurity, UI/UX, etc.).

## How It Works
  > User inputs a learning goal (e.g., “I want to become a UI/UX designer”).
   IBM Granite LLM processes the request.
   Vector Index retrieves curated content from uploaded PDFs & trusted sources.
   Agent generates a personalized, step-by-step learning roadmap.


##  How to Run or Deploy
    > Log in to IBM Cloud Lite → https://cloud.ibm.com
      Launch watsonx.ai Studio
      Create a new AI Agent named Personalized Path
      Upload curated learning PDFs to a Vector Index
      Choose tools for web search (Google, Wikipedia, DuckDuckGo, etc.)
      Configure agent instructions and topics (with polite redirection for off-topic queries)
      Test in the preview panel
      Deploy via web snippet, Streamlit app, or custom web UI
